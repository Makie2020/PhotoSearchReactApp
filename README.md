Search Photo App

The Search Photo App is the third project that is part of the training at OXYGEN Academy
This app is created with Create-React-App, React-Router, Redux (with redux template). The design was made in Figma, 

It is a dashboard to download and manage images. You have the option to save you favourite images in a Gallery.
